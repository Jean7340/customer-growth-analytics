Facebook Advertising Performance Analysis

Author

Jean Liu

Project Overview

This project analyzes Facebook advertising performance across 4,431 ads to identify which campaign attributes are associated with higher click efficiency.

The core business question is:

Which advertising strategies generate more clicks per dollar, and how should budget and creative strategy be adjusted?

The analysis evaluates campaign performance across placement, ad type, creative message, retail category, and audience age. It combines exploratory analysis with regression modeling to separate surface-level performance differences from statistically significant performance drivers.

Show Code 
# Load raw advertising dataset
facebook <- read_csv("../data/Facebook Data.csv")

facebook_clean <- facebook |>
  select(-`...1`) |>
  mutate(
    date = as.Date(date, "%m/%d/%Y"),

    adType = factor(adType),
    placement = factor(placement),
    category = factor(category),
    keywords = factor(keywords),
    body = factor(body)
  )

Regression Modeling

Descriptive statistics provide useful comparisons across campaign attributes, but they cannot isolate the independent effect of each advertising characteristic.

To better understand which factors are associated with advertising performance, a multiple linear regression model is estimated using Clicks per Dollar as the dependent variable.

Show Code 
model_1 <- lm(
  clickPerDollar ~
    adType +
    placement +
    keywords +
    body +
    category +
    ageMean,
  data = facebook_clean
)

model1_results <- tidy(model_1)
Table 7. Regression Model Summary

Interaction Analysis

To explore whether audience age influences campaign performance differently across retail categories, an interaction model was estimated.

model_2 <- lm(

    keywords +
    body +

Key Findings

The analysis identified several consistent patterns associated with stronger Facebook advertising performance.

  1. Mobile advertisements consistently generated more clicks per advertising dollar than desktop campaigns.

  2. Photo posts outperformed link posts, suggesting that visual creatives attract stronger customer engagement.

  3. Engagement-focused messages encouraging users to “Like” the brand achieved the highest click efficiency.

  4. Advertising performance differed across retail categories, indicating that campaign effectiveness depends on product context.

  5. Audience age showed different relationships with performance across retail categories, suggesting that demographic targeting should be adjusted by campaign type.

Business Recommendations

Based on the analysis, the following recommendations could improve Facebook advertising performance.

  1. Allocate more advertising budget to mobile placements, where campaigns consistently generated higher click efficiency.

  2. Use photo-based creatives whenever appropriate, as they outperformed link posts across both click-through rate and clicks per dollar.

  3. Continue testing engagement-focused messages, especially calls-to-action encouraging users to interact with the brand.

  4. Tailor campaign strategy by retail category instead of using the same creative approach across all products.

  5. Consider audience age when defining targeting strategies, since its relationship with performance varies across retail categories.
